ユーザーから渡された商談情報をもとに、パイプラインレポートを更新する。

## 入力

$ARGUMENTS

## 手順

1. `index.html` を読み込む
2. CLAUDE.md のビジネスルール（月別帰属・KPIウォーターフォール・移行率計算・ツールチップ）に従い更新する
3. 更新対象:
   - 該当月タブの商談テーブル（行の追加・ステージバッジ変更・備考更新）
   - KPIカードの数値（累積ルールに従い再計算）
   - 移行率・転換率の再計算
   - ツールチップ内の企業リスト
   - 「全体」タブ（全月合算で再計算）
   - 新しい月タブが必要ならempty-stateを実データに置き換え
4. `executive_summary.html` にコピー
5. git commit & push
6. 変更サマリーを報告
